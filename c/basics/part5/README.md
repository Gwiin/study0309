# Part 5

구조체와 문자열, 파일 단위 문자 입출력을 익히는 파트입니다.

## 다루는 내용

- `enumExample.c`: 열거형
- `struct.c`, `struct2.c`, `struct3_twoint.c`, `structArray.c`: 구조체
- `unionExample.c`: 공용체
- `stringExample.c`, `string3.c`: 문자열 처리
- `charGetPut*.c`: 문자 단위 파일 입출력
- `selfReference.c`: 자기 참조 구조체
- `date/`, `twoDouble/`: 구조체 기반 응용 실습

## 목표

- 사용자 정의 자료형을 읽고 설계하는 감각 익히기
- 문자열과 파일 입출력을 함께 다루는 기초 다지기
