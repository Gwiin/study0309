# Part 1

C 언어 입문 파트입니다. 가장 기본적인 출력, 자료형 표현, 연산자와 빌드 흐름을 익히는 데 초점을 둡니다.

## 다루는 내용

- `hello.c`: 가장 기본적인 프로그램 구조와 `main`
- `literal.c`, `limit.c`: 리터럴, 출력 형식, 자료형 범위
- `compare.c`, `compare2.c`, `oddEven.c`: 비교 연산과 조건 표현
- `increment.c`, `sizeof.c`: 증감 연산과 메모리 크기 개념
- `fourbit.c`, `fourbit2.c`: 비트 연산 기초
- `makefile`, `makefile_temp`, `makefile_temp2`: Makefile 연습

## 목표

- C 프로그램이 어떻게 컴파일되고 실행되는지 익히기
- 기본 자료형과 출력 형식을 구분하기
- 연산자와 비트 표현을 코드로 확인하기
