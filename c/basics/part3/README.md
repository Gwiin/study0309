# Part 3

포인터와 함수 중심의 파트입니다. C 언어에서 메모리를 다루는 핵심 감각을 익히는 구간입니다.

## 다루는 내용

- `add.c`, `swap.c`: 함수 분리와 매개변수 전달
- `localVariable.c`: 지역 변수와 수명
- `pointer1.c`, `pointer2.c`, `pointer3.c`: 포인터 기본
- `pointerAndArray.c`, `pointerAndArray2.c`: 배열과 포인터 관계
- `pointer5_functionpointer.c`: 함수 포인터
- `pointer6_voidpointer.c`: `void *`
- `bubbleSorting.c`, `selectionSorting.c`, `qsort.c`: 정렬 알고리즘과 비교 함수

## 목표

- 주소와 값의 차이를 분명히 이해하기
- 포인터를 함수와 배열에 연결해서 읽을 수 있게 만들기
- 정렬 예제를 통해 함수 설계와 자료 접근 방식 익히기
