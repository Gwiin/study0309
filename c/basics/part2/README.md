# Part 2

조건문, 반복문, 배열을 다루는 파트입니다. 입력값 판단과 반복 처리의 흐름을 익히는 데 적합합니다.

## 다루는 내용

- `passFail.c`, `scoreGrade.c`, `scoreGradeSwitch.c`: `if`, `switch`
- `oneToTenFor.c`, `one2TenWhile.c`, `one2TenSum.c`: `for`, `while`
- `dice.c`, `dice2.c`, `lotto.c`: 난수와 반복
- `arrayInit.c`, `sumArray.c`, `sumMatrix.c`: 1차원, 2차원 배열
- `star.c`, `star2.c`, `xstar.c`: 반복문 패턴 출력

- 제어문 흐름을 코드로 읽고 직접 수정해 보기
- 배열 초기화와 반복 처리의 관계 이해하기
- 작은 문제를 반복문으로 분해하는 감각 익히기
