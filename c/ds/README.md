# Data Structures

자료구조 구현 연습을 모아둔 폴더입니다. C 기초 문법을 바탕으로 스택, 큐, 리스트를 직접 구현해 보는 흐름으로 구성되어 있습니다.

## 구성

| 폴더 | 내용 | 링크 |
| --- | --- | --- |
| `list/` | 연결 리스트 기초 실습 | [README](./list/README.md) |
| `stack/` | 여러 방식의 스택 구현 | [README](./stack/README.md) |
| `queue/` | 큐 구현과 변형 예제 | [README](./queue/README.md) |

## 학습 포인트

- 구조체와 포인터를 실제 자료구조 구현에 적용합니다.
- `main.c`와 구현 파일을 분리하면서 모듈 구성을 연습합니다.
- CMake 기반 빌드 예제가 함께 있어 빌드 구조를 익히기 좋습니다.
