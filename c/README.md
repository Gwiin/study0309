# C

C 언어 학습 내용을 기초 문법, 자료구조, 프로젝트로 나누어 정리한 영역입니다.

## 구성

| 폴더 | 내용 | 링크 |
| --- | --- | --- |
| `basics/` | C 문법, 배열, 포인터, 구조체, 파일 입출력 | [README](./basics/README.md) |
| `ds/` | 스택, 큐, 리스트 자료구조 구현 | [README](./ds/README.md) |
| `projects/` | C로 만든 미니 프로젝트 | [README](./projects/README.md) |

## 학습 흐름

1. `basics/`에서 문법과 메모리 개념 익히기
2. `ds/`에서 구조체와 포인터를 자료구조로 확장하기
3. `projects/`에서 여러 파일을 묶은 프로그램으로 연결하기
