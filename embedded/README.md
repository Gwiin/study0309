# Embedded

하드웨어 제어와 장치 간 통신을 중심으로 한 임베디드 학습 영역입니다.

## 구성

| 폴더 | 내용 | 링크 |
| --- | --- | --- |
| `atmega128/` | AVR ATmega128 실습 | [README](./atmega128/README.md) |
| `pico/` | Raspberry Pi Pico / Pico 2 W 실습 | [README](./pico/README.md) |
| `network/` | MQTT와 네트워크 예제 | [README](./network/README.md) |

## 학습 흐름

1. `atmega128/`에서 GPIO, 타이머, 인터럽트, 센서 제어 익히기
2. `pico/`에서 보드 환경과 네트워크 연결 실습 확장하기
3. `network/`에서 MQTT 발행 흐름과 통신 예제를 정리하기
