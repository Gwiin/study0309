# C++

앞으로 진행할 C++ 학습 공간입니다. 기존 C 학습 흐름과 이어서, 문법 정리와 실습 코드를 이 폴더 아래에 누적합니다.

## 폴더 구성

| 폴더 | 내용 | 링크 |
| --- | --- | --- |
| `notes/` | 문법, 개념, 수업 요약 | [README](./notes/README.md) |
| `practice/` | 실습 코드와 예제 | [README](./practice/README.md) |

## 운영 방식

- 새 수업을 들은 날에는 먼저 `notes/`에 핵심 개념을 정리합니다.
- 실습 코드는 `practice/`에 날짜나 주제 기준으로 추가합니다.
- C와 비교해 중요한 차이점은 `README`나 노트에 따로 적어두면 복습에 도움이 됩니다.
